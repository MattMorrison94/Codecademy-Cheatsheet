Semantic HTML

Tag Name Description
header Header section The header element represents a container for introductory content or a set of navigation links.
nav Navigation section The nav element defines a set of navigation links.
section Section The section element defines a section in a document. This is the semantic equivalent of a div.
article Article The article element specifies independent, self-contained content.
aside Aside The aside element defines defines some content aside from the content it is placed in (like a sidebar).
footer Footer Defines a footer for a document or section.
